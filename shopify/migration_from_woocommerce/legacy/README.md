# Legacy scripts — reference only

Retired Python scripts that seeded the Shopify dev store for the 2026-07-14 client demo/getting-started session (`pull_demo_source.py` pulled a slice of the live WooCommerce data, `push_demo_seed.py` loaded it, `get_admin_token.py` minted the offline Admin API token).

They were rushed, one-off tooling. **Do not run or extend them.** The real migration is done by the Node.js tool in `../migration-tool/` (see `docs/migration-tool-prd.md` / `docs/migration-tool-plan.md`). Kept for reference; delete only at project end and only after asking Bassam.
